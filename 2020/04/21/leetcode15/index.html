三数之和

题目

题解

例如[-1, 0, 1, 2, -1, -4]

  1. 先排序，从小到大[-4 ,-1 ,-1 ,0 ,1 ,2 ]
  2. 选取一个索引i的值nums[i]当初始值a,其他算从num[i+1]为b和排序好的数组末尾num[n-1]为c
    如果三者=0，这存入，bc均移动一位；如果>0,则c向后退一位<0则b前进一位；
    选i=0; a = -4,b= -1,c = 2, a+b+c<0;b前进b=0,c=2,a+b+c<0… 无解；
    选i=1; a = -1,b=-1,c = 2 ,a+b+c = 0,记录[-1, -1, 2];b = -1;c = 1,记录[-1, 0, 1]…
    选i=2;和情况1相似，跳过
    选i=3; a +b +c>0
    综合结果为：[[-1, 0, 1],[-1, -1, 2]]
