  • 关于hexo 和 jekyll

    主要是jekyll需要配置的文件有点多，我个人也比较菜（lan），之前的jekyll虽然成功了，但是操作步骤还是挺多的。。

  • github配置blog

  1. 新建一个和你的用户名相同的名字，用来当做域名{youName}.github.io,格式不限
  2. 添加你的仓库配置文件，随便选一个就好，我java用的多就选了java
  3. 证书，选Apache2.0吧
  4. 忽略我的io1中的1，选择设置

  5. 选择master分支，选择主题
  6. 随便选一个吧
  7. 成功
  8. 添加一个readme并且添加index.html(默认首页名)


    9.成功

    这样你可以访问你的博客了！地址是{youName}.github.io

  • 安装hexo

  1. 默认你的git和nodejs已经安装好了
    cmd查询一下你的版本

  2. 在空白处右击选择Git Bash Here，打开npm界面（这个终端的剪贴快捷键是shift+ins）

  3. 安装hexo(可能需要梯子)
    npm install -g hexo
    安装的时候可能会没有反应，等等就好了，也可以考虑换源，不过博主等了几分钟，看了会视频就好了

  4. 依赖，插件什么的统统装上吧,下面两个是主要的

  5. 建一个blog的文件夹，右击打开npm,”$” 已经存在
